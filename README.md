# 全能开发者能力模型

> AI时代独立全包软件项目的能力评估工具

## 简介

本项目是一个交互式的开发者能力模型可视化工具，涵盖AI时代独立承接软件项目所需的八大核心能力维度。

## 功能特性

### 1. 八大能力维度展示
- **技术硬能力** - 前后端开发、数据库、部署运维
- **产品&需求** - 需求分析、原型设计、AI辅助需求处理
- **项目管理** - 排期规划、风险预判、成本控制
- **测试&质量** - 单元测试、边界测试、AI辅助测试
- **沟通&商务** - 客户对接、方案输出、需求引导
- **架构&设计** - 轻量化架构、代码规范、AI友好设计
- **心态&素养** - 自律、学习力、抗压能力
- **AI应用能力** - Agent编码、AI需求处理、AI文档生成

### 2. 可视化展示
- 中心枢纽动画展示8大能力环绕
- 项目全流程角色图谱
- AI全链路辅助流程图

### 3. 能力评估雷达图
- 输入姓名
- 为8大能力维度打分（1-10分）
- 生成个人雷达图
- 自动计算总分（满分80分）

### 4. 交互体验
- 鼠标悬停显示能力名称
- 卡片悬浮发光效果
- 响应式设计，支持移动端

## 使用方法

1. 直接在浏览器中打开 `capability-model.html`
2. 浏览八大能力维度的详细说明
3. 滚动到页面底部的"能力评估打分"区域
4. 输入姓名，调整各项能力分数
5. 点击"生成雷达图"查看结果
